## рџљЂ Installation

### 1. Clone the repository

```bash
git clone <repository-url>
cd AI_Multiverse_2
```

### 2. Create a virtual environment

```bash
python -m venv .venv
```

### 3. Activate the virtual environment

**Windows**

```bash
.venv\Scripts\Activate.ps1
```

**macOS / Linux**

```bash

```

### 4. Install dependencies

```bash

```

```env

```

