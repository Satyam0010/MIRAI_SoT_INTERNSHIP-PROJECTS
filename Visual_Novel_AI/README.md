



# 📖 AI Multi-Modal Visual Novel

An interactive AI-powered **Choose Your Own Adventure** Visual Novel built using **Google Gemini**, **Pollinations AI**, **gTTS**, and **Streamlit**.

This project was developed as the **Capstone Mini Project** for the **MirAI School of Technology – Virtual Summer Internship 2026 (AI Builder Track)**.

---

## 🚀 Features

- 🎭 Choose your preferred **Story Genre**
- 🎨 Select an **Art Style**
- 🤖 AI-generated interactive story using **Google Gemini**
- 🖼️ Dynamic scene generation with **Pollinations AI**
- 🔊 AI narration using **Google Text-to-Speech (gTTS)**
- 🎮 Dynamic choice buttons generated by AI
- 💾 Stateful story progression using Streamlit Session State
- 📦 Structured JSON responses for reliable parsing
- ⚠️ Graceful error handling with `try...except`
- 🔄 Infinite story continuation based on user choices

## 🎮 How It Works

1. Select a Story Genre.
2. Select an Art Style.
3. Click **Start Story**.
4. Gemini generates:
   - Story narration
   - Image prompt
   - Interactive choices
5. Pollinations AI creates the scene image.
6. gTTS converts the story into speech.
7. Click any generated option to continue the adventure.

The story continues indefinitely based on your choices.
